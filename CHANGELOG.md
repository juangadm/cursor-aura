# Changelog

## [0.1.0] - 2026-02-04
### Added
- Initial release
- React component (`<Aura />`)
- Vanilla JS API (`Aura.init()`, `setColor()`, `destroy()`)
- 5 cursor types: default, pointer, grab, grabbing, text
- CSS variable support for dynamic theming
- Accessibility: reduced motion, touch device detection
